Projetos

Prevendo a emissão de CO² dos automóveis

Este projeto apresenta a construção de um modelo preditivo para estimar a quantidade de CO2 emitida por automóveis. O modelo é baseado em um grande banco de dados de uma concessionária canadense que descreve várias características de centenas de automóveis. O objetivo é criar um modelo preciso e confiável que possa ser usado para prever as emissões de CO2 de diferentes tipos de automóveis. O modelo foi construído usando uma combinação de técnicas estatísticas, incluindo análise de regressão múltipla e métodos de transformação de dados.
