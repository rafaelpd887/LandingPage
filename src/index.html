Projetos

Prevendo a emissão de CO² dos automóveis

Este projeto apresenta a construção de um modelo preditivo para estimar a quantidade de CO2 emitida por automóveis. O modelo é baseado em um grande banco de dados de uma concessionária canadense que descreve várias características de centenas de automóveis. O objetivo é criar um modelo preciso e confiável que possa ser usado para prever as emissões de CO2 de diferentes tipos de automóveis. O modelo foi construído usando uma combinação de técnicas estatísticas, incluindo análise de regressão múltipla e métodos de transformação de dados.

Estimando as vendas do varejo através de uma série temporal

A intenção aqui foi criar um modelo preciso através de uma série temporal do IBGE que possa ser usado para prever o volume das vendas futuras no varejo do estado de Minas Gerais. A série temporal em questão foi analisada e modelada utilizando técnicas estatísticas, incluindo modelagens ARIMA e ETS, afim de capturar diferentes aspectos da mesma. Ao final, foi possível validar a qualidade das previsões por meio de métricas de desempenho, como o erro médio absoluto (MAE).

Um modelo logístico para concessão de empréstimo

Criação de um modelo logístico que seja capaz de estimar a probabilidade de aprovação de empréstimo com base nas características dos solicitantes. O modelo foi feito com base em um conjunto de dados que contém informações dos solicitantes, juntamente com a indicação se o empréstimo foi aprovado ou não para cada um deles. Após a construção do modelo, foi realizada a avaliação da sua performance, utilizando métricas como a acurácia, precisão, e a curva ROC (Receiver Operating Characteristic).

Segmentação de clientes com Análise de Clusters

Através de um banco de dados com informações de compra dos clientes de um distribuidor atacadista Português, iremos aplicar uma análise de cluster com o objetivo de segmentar esses clientes em diferentes grupos (clusters). Rankeando RMs através da Análise Fatorial

Com o rápido crescimento das grandes cidades brasileiras, o fenômeno da conurbação tem levado à formação de diversas regiões metropolitanas (RMs) em todo o território nacional. O Instituto Brasileiro de Geografia e Estatística (IBGE) disponibiliza um extenso banco de dados com informações sobre o nível de desenvolvimento das principais regiões metropolitanas do Brasil. A fim de simplificar essas informações e obter uma visão mais abrangente, faremos uma Análise Fatorial por Componentes Principais. Essa análise nos permitirá reduzir a complexidade do banco de dados em fatores significativos, que serão utilizados para classificar as regiões metropolitanas de acordo com seu grau de desenvolvimento.

Análise de correpondência para pacientes cardíacos

A análise de correspondência é uma técnica que permite avaliar a associação entre variáveis qualitativas e entre suas categorias. Ela também é capaz de condesar as informações dessas variáveis em valores númericos que podem ser plotados em diversos tipos de gráficos, ou até mesmo usados em outros tipos de modelos analíticos. Nesse projeto, analizaremos um banco de dados com informações cariovasculares de centenas de pacientes médicos para compreendermos quais carcterísticas podem estar associadas a doenças cardíacas.
