Projetos

Prevendo a emissão de CO² dos automóveis

Este projeto apresenta a construção de um modelo preditivo para estimar a quantidade de CO2 emitida por automóveis. O modelo é baseado em um grande banco de dados de uma concessionária canadense que descreve várias características de centenas de automóveis. O objetivo é criar um modelo preciso e confiável que possa ser usado para prever as emissões de CO2 de diferentes tipos de automóveis. O modelo foi construído usando uma combinação de técnicas estatísticas, incluindo análise de regressão múltipla e métodos de transformação de dados.

Estimando as vendas do varejo com uma série temporal

A intenção aqui é criar um modelo preciso através de uma série temporal que possa ser usado para prever o volume das vendas futuras do estado de Minas Gerais. A série temporal foi analisada e modelada utilizando técnicas estatísticas, incluindo modelos ARIMA (AutoRegressive Integrated Moving Average), GARCH (Generalized Autoregressive Conditional Heteroskedasticity) e ETS (Error-Trend-Seasonality) para capturar diferentes aspectos da mesma. Ao final, foi possível validar a qualidade das previsões por meio de métricas de desempenho, como o erro médio absoluto (MAE).

Um modelo logístico para concessão de empréstimo

Criação de um modelo logístico que seja capaz de estimar a probabilidade de aprovação de empréstimo com base nas características dos solicitantes. O modelo foi feito com base em um conjunto de dados que contém informações dos solicitantes, juntamente com a indicação se o empréstimo foi aprovado ou não para cada um deles. Após a construção do modelo, foi realizada a avaliação da sua performance, utilizando métricas como a acurácia, precisão, e a curva ROC (Receiver Operating Characteristic).
